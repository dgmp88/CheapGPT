# CheapGPT

Cheaper chat GPT, using your own API key.

Deployed [here](https://dgmp88.github.io/CheapGPT/).

# Built with Svelte

[`create-svelte`](https://github.com/sveltejs/kit/tree/master/packages/create-svelte).

## Developing

```bash
npm run dev
```

## Building

```bash
npm run build
```
